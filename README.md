# SKYNET Demo Foundation

Minimal Next.js App Router starter for a SkyNet GB visual pitch concept.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- `src/` directory layout

## Development

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run start
```

## Deployment

This project is ready to deploy on Vercel with zero additional configuration.
